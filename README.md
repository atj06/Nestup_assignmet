A Python-based 3D visualization system using Matplotlib to model a cabinet with internal planks, generating accurate spatial structures, dimensions, and labeled components for clear representation and analysis.

Output
The program generates a 3D visualization of a cabinet structure with internal planks, accurately placed using given dimensions and coordinates.

Approach
- Defined cabinet and wall dimensions
- Used a custom function to draw 3D boxes
- Placed planks based on structured input data
- Added labels and visual elements for clarity

Libraries Used
- numpy
- matplotlib

1. Install dependencies
2. Run the Python file
